================================================================================
                    GitHub Actions Auto Build Setup
================================================================================

STEP 1: Create GitHub Repository
--------------------------------------------------------------------------------
1. Go to: https://github.com/new
2. Repository name: HiddenAlbumToggle
3. DO NOT check any initialization options
4. Click "Create repository"
5. Copy the repository URL (e.g., https://github.com/username/HiddenAlbumToggle.git)


STEP 2: Push Code to GitHub
--------------------------------------------------------------------------------
Choose ONE method:

METHOD A - PowerShell (Recommended for Windows):
   1. Right-click "push-to-github.ps1"
   2. Select "Run with PowerShell"
   3. Enter your repository URL when prompted

METHOD B - Command Prompt:
   1. Double-click "push-to-github.bat"
   2. Enter your repository URL when prompted

METHOD C - WSL/Linux:
   1. Open terminal
   2. Run: bash push-to-github.sh
   3. Enter your repository URL when prompted


STEP 3: Watch the Build
--------------------------------------------------------------------------------
1. Go to your GitHub repository
2. Click "Actions" tab
3. Wait for build to complete (2-5 minutes)


STEP 4: Download .deb File
--------------------------------------------------------------------------------
1. Click on the successful workflow run (green checkmark)
2. Scroll to bottom "Artifacts" section
3. Download "HiddenAlbumToggle-deb"
4. Extract the ZIP to get .deb file


STEP 5: Install on iOS Device
--------------------------------------------------------------------------------
1. Transfer .deb to your iOS device
2. Install using Filza or package manager
3. Respring: killall -9 SpringBoard


================================================================================
                              Troubleshooting
================================================================================

Q: Script shows encoding errors?
A: Use PowerShell version (push-to-github.ps1) instead

Q: Git not found?
A: Install Git from https://git-scm.com/downloads

Q: Push failed - authentication error?
A: You may need to setup GitHub credentials or SSH keys

Q: Build failed on GitHub?
A: Check Actions logs for detailed error messages
